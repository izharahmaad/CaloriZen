CaloriZen is a **modern, easy-to-use calorie planner app** built with React Native and Expo. It helps users calculate and track their daily calorie needs whether the goal is to **maintain, lose, or gain weight**. With a clean design, smooth animations, and smart features, CaloriZen makes calorie management **simple, intuitive, and visually appealing**. Perfect for fitness enthusiasts, beginners, and anyone looking to improve their health.

cd CaloriZen
npm install
npx expo start
# Test Build (APK)

# Production Build (AAB for Play Store)
eas build -p android --profile production
Open the project with Expo Go on Android or iOS.
